年末年始 学習継続支援｜講師スケジュール管理ツール

概要
本アプリは、年末年始休校期間の学習継続支援施策として企画・制作した、講師（サポート役）の予定を可視化する管理ツール型Webアプリケーションです。
年末年始の休校期間中は、メンターや講師への質問・相談ができず、学習が孤立しやすくなります。そこで本アプリでは、「今、誰が対応可能なのか」を受講生が一目で確認できる環境を提供することを目的としています。
HTML・CSS・JavaScriptのみで構成し、ブラウザの localStorage を用いてデータを管理しています。

アプリURL
https://358tech-camp.github.io/project-folder7/list.html

制作背景・目的
テックキャンプでは年末年始に休校期間が設けられており、以下の期間は講師・メンターへの質問やレビュー対応が停止します。
［休校期間］2025/12/31（水）〜2026/1/3（土）
この期間中も学習を継続したい受講生が、講師の稼働予定やサポート状況を把握しやすくすることを目的として、本ツールを制作しました。
* メンター・講師への質問や相談ができない
* 学習のつまずきを解消できず、モチベーションが低下しやすい
* 一人での学習が続き、挫折リスクが高まる
これらの課題に対し、講師や卒業生サポート役の稼働予定を可視化することで、受講生が安心して学習を継続できる環境を作りたいと考え、本ツールを制作しました。
本アプリの目的は以下の通りです。
* 年末年始でも学習を止めずに継続できる仕組みを提供する
* 講師・サポート役の対応可能時間を一覧で確認できるようにする
* 学習者の孤独感を軽減し、安心感を与える

主な機能
* 講師プロフィールの登録・編集・削除
* プロフィール画像の登録・プレビュー表示
* 講師ごとのスケジュール登録（30分単位）
* 全講師スケジュールの一覧表示
* 講師選択によるスケジュール絞り込み
* 講師名ホバー時のプロフィールツールチップ表示
* 本人判定による編集権限制御（本人のみ編集可能）

使用技術
* HTML5
* CSS3
* JavaScript（Vanilla JS）
* localStorage
※ サーバーサイド（Ruby / Ruby on Rails / DB）は使用していません。

画面構成
* 講師スケジュール一覧画面 全講師のスケジュールを日付・時間帯ごとに一覧表示
* 講師プロフィール登録画面 氏名・専門分野・自己紹介・画像の登録、編集、削除
* 講師スケジュール編集画面 講師本人のみがスケジュールを登録・変更可能

工夫した点
* 本人判定ロジックを実装し、編集権限を制御
* スケジュールを日付×時間帯でグループ化し、視認性を向上
* ツールチップで講師の詳細情報を即座に確認できるUI設計
* 管理ツールとして直感的に操作できる画面構成

今後の改善・拡張案
* Ruby on Rails を用いたサーバーサイド実装
* DB（PostgreSQL / MySQL）によるデータ永続化
* ユーザー認証機能の追加
* 管理者・講師・閲覧者などの権限分離
* Discordサポート企画との連携強化

補足
本アプリはRailsでの制作ではないため、テックキャンプの「オリジナルアプリ」要件には該当しませんが、 年末年始の学習継続という課題に対する解決策として企画・実装した自主制作アプリとして、ポートフォリオに掲載しています。
