# 年末年始学習継続支援｜講師スケジュール管理ツール

## アプリケーション名

年末年始学習継続支援｜講師スケジュール管理ツール

## アプリケーション概要

年末年始の休校期間中に、講師・サポート役の稼働予定を可視化することで、受講生が安心して学習を継続できるよう支援するWebアプリケーションです。

講師プロフィールや対応可能な時間帯を一覧で確認でき、学習中の孤立や不安を軽減することを目的としています。

## 利用方法

* 講師スケジュール一覧画面で、全講師の稼働予定を確認できます
* 「講師登録」画面から、講師プロフィールを登録できます
* 登録した講師本人のみが、スケジュールの登録・編集・削除を行えます

## アプリケーションを作成した背景

テックキャンプでは年末年始に休校期間があり、その間は講師メンターへ質問・相談ができません。
その結果、学習中につまずいても不安を抱えたまま学習を続けてしまい、モチベーション低下や挫折につながる課題があると感じました。

本アプリは、講師や卒業生サポート役の対応可能時間を可視化することで、
「今は質問できなくても、この時間に相談できる人がいる」という心理的な安心感を提供し、学習の継続を後押しすることを目的として制作しました。

## 実装した機能

* **講師プロフィール登録・編集・削除機能**
  https://gyazo.com/b85acd8c7e7a039d3067ac669f5298ae
  
  講師の専門分野や自己紹介を登録でき、受講生が「誰に相談できるのか」を事前に把握できます。

* **プロフィール画像の登録・プレビュー表示**
  https://gyazo.com/fe5ff4dc36d70152b380054245e783a6
  
  顔が見えることで、心理的な距離を縮め、相談へのハードルを下げる工夫をしています。

* **講師ごとのスケジュール登録機能（30分単位）**
  https://gyazo.com/639e07c6fbc9ddf4733e8768207e53ef
  
  対応可能時間を細かく管理でき、実際のサポート運用を想定した設計にしています。

* **全講師スケジュール一覧表示機能**
  https://gyazo.com/74bd4544127b4bf65cc5aa8ca191f46b
  
  受講生が一目で全体状況を把握できるよう、日付 × 時間帯で可視化しています。

* **講師選択によるスケジュール絞り込み**
  https://gyazo.com/7af8755eb4596c84cb3006590bd821b8
  
  特定の講師に絞って確認でき、実用性を高めています。

* **本人のみ編集可能な権限制御**
  https://gyazo.com/527230b54fbcbb069df8ced9dfce5d8c
  
  登録者本人だけが情報を編集できるようにし、データの信頼性を担保しています。

## 実装予定の機能

* Ruby on Rails を用いたサーバーサイド実装
* DB（PostgreSQL / MySQL）によるデータ永続化
* ユーザー認証機能の追加
* 管理者・講師・閲覧者の権限分離

## 工夫したポイント

* **課題から逆算した機能設計**
  「学習継続の不安を減らす」という目的から、まず必要な情報は何かを整理し、プロフィール表示・スケジュール可視化に機能を絞りました。

* **サーバー不要で動作する構成**
  localStorageを利用し、フロントエンドのみで完結する構成とすることで、設計・実装・公開までを一貫して行いました。

* **本人判定ロジックによる権限制御**
  JavaScriptで本人のみが編集可能となるロジックを実装し、将来的な認証機能追加を見据えた設計にしています。

* **UI/UXを意識した情報設計**
  日付 × 時間帯でのスケジュール管理や、ツールチップ表示など、管理ツールとして直感的に操作できることを意識しました。

## 改善点（今後の実装計画）

* **サーバーサイド（Ruby on Rails）の導入**
  データの永続化や複数端末での利用を可能にするため、Rails + DB での再設計を予定しています。

* **ユーザー認証機能の実装**
  講師・閲覧者を明確に分離し、より厳密な権限制御を行えるようにします。

* **モバイル対応の強化**
